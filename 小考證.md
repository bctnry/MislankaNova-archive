# 小考證

我一小時後有考試。所以寫得簡單點。

這篇的內容來自我之前寫過的一篇「考證」文章。那篇文章在知乎發過，兩天後發現有重大紕漏，就刪掉了。現在把沒有紕漏的部分發上來。也許算不上考證。但至少是一點創作。

＊＊＊

+ 秘封組專輯《燕石博物誌》之英文副標題，是「Dr. Latency's Freak Report」，直譯爲中文，可以作「隱祕博士的可怖報告」。
+ 這 一標題讓我想到了博爾赫斯的小說《布羅迪報告》。 這部小說的西班牙語原文標題是「El informe de Brodie」，沒有「博士」一詞。但是，這篇小說最早的英文譯本，將其翻譯成「Dr. Brodie's Report」，加上了「博士」一詞。這麼修改之後，其標題結構與《燕石》之英文副標題有幾分相像。
+ 《燕石》的故事主旨，是秘封組兩人編輯一本「博物志」，當中記載有「兩人見到的不可思議的世界」的事情。
+ 《布羅迪報告》則以「作者」發現一份手稿開始。手稿中記載有傳教士大衛・布羅迪對南美洲一支原住民部落的觀察報告。可以當作是在「窺視」異世界。
+ 「布羅迪」是來自蘇格蘭的姓氏。
+ 小泉八雲之父是駐紮在伊奧尼亞群島的愛爾蘭士兵。
+ 在十八世紀的愛丁堡，有一位本名威廉・布羅迪的家具匠，同時也是鎖匠。他白天做一份正人君子的工作。到了晚上，就靠他修鎖時積累的經驗來行竊。是一位白天和黑夜有着兩幅面貌的人。
+ 威廉・布羅迪的故事啓發了出身愛丁堡的作家羅伯特・路易斯・斯蒂文森。斯蒂文森的名作《化身博士》就是受此影響而寫。
+ 博爾赫斯曾在一本書的後記中言明斯蒂文森對其的影響。
